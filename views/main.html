Usage

We use bower for dependency management. Add 

dependencies: {
    "ui-rect-select": "latest"
}

To your bower.json file. Then run

bower install

This will copy the ui-rect-select files into your components folder, along with its dependencies.

Add the css:

<link rel="stylesheet" href="bower_components/ui-rect-select/dist/rect-multi-select.css">

Load the script files in your application:

<script src="bower_components/jquery/dist/jquery.js"></script>
<script src="bower_components/angular/angular.js"></script>
<script src="bower_components/ui-rect-select/dist/rect-multi-select.js"></script>

Add the uiSelect module as a dependency to your application module: 

var myAppModule = angular.module('MyApp', ['uiSelect'])

Add the css selectable-container and apply the directive rect-multi-select to your form elements:

<div class="selectable-container"
    rect-multi-select="{'selector': '.photo'}"
    rect-move-select="{{'{{moveSelect'+'}'+'}'}}"
    rect-selector="photo.actionSelected">
</div>

Options

selector

To be selected element's css selector

rect-multi-select="{'selector': '.photo'}"
<div ng-repeat="photo in photos"
    ng-controller="PhotoCtrl"
    class="photo">
 <img ng-src="{{'{{baseUrl'+'}'+'}{{photo.name'+'}'+'}'}}>
</div>

rect-move-select

Real time select elements on mouse move event, moveSelect is an boolean variable of $scope.

<div class="selectable-container"
    rect-multi-select="{'selector': '.photo'}"
    rect-move-select="{{'{{moveSelect'+'}'+'}'}}"
    rect-selector="photo.actionSelected">
</div>

rect-selector

The variable name of indicator for selected element in their scope

<div class="selectable-container"
    rect-multi-select="{'selector': '.photo'}"
    rect-move-select="{{'{{moveSelect'+'}'+'}'}}"
    rect-selector="photo.actionSelected">
</div>

Example

<div class="selectable-container"
    rect-multi-select="{'selector': '.photo'}"
    rect-move-select="{{'{{moveSelect'+'}'+'}'}}"
    rect-selector="photo.actionSelected">
    <div class="photo-wall photo-selectable-container">
      <div ng-repeat="photo in photos"
          ng-controller="PhotoCtrl"
          class="photo"
          ng-class="{'selected': photo.actionSelected}">
        <img ng-src="{{'{{baseUrl'+'}'+'}{{photo.name'+'}'+'}'}}>
      </div>
    </div>
</div>
